HTML-CSS KULLANARAK WEB SİTESİ

HTML-CSS İle Neler Yapıyorum?


İşaretleme dili olan Html, web sayfalarının hazırlanmasında kullanılan sistemdir. Bir programlama dili olmayan Html bilgisayarlarımızda kullandığımız web sitelerinin oluşturulmasında kullanılır. Chrome, Fİrefox ve İnternet Explorer gibi tarayıcılar html kodlarını işleyerek bu kodları web sayfasına dönüştürür. Bende Web Tasarım Projesinde verilmiş olan kişisel web sitemi oluştururken HTML kullanıyorum ve Html kodlarımı Visual Studio Code'de yazıyorum.

CSS nedir? Neden Kullanırız?


CSS en temel haliyle HTML etiketlerimizin görsel açından biçimlendirilmesini (Renk, yazı şekli, arka plan rengi, genişlik, yükseklik, pozisyon durumu vs) sağlar. CSS’in sahip olduğu seçiciler vardır. Bunlar id ve class olarak ikiye ayrılır. HTML etiketlerinde tanımladığımız seçici isimleriyle birlikte CSS kullanarak etiketleri biçimlendiririz.


CSS’i HTML ile oluşturduğumuz iskelet yapılarını görsel açıdan daha iyi hale getirmek için kullanırız.

Projeme CSS Kodlarını Nasıl Dahil Ettim?


Yazmış olduğum css dosyasını html kodlarıma nasıl dahil ettim? Öncelikle olarak proje dosyamın içerisinde style.css uzantılı bir dosya oluşturdum. Ve html kodlarımı yazmış olduğum index.html uzantılı dosyamın içerisinde yer alan header alanına yanda göründüğü üzere link rel="stylesheet" type="text/css" href="style.css" yazarak css dosyamı dahil ettim.
